# Stellar Asset Management Protocol

Quickstart (TypeScript / Node):

1. Install dependencies

```powershell
npm install
```

2. Run the testnet demo (funds a generated keypair via Friendbot and prints balances)

```powershell
npx ts-node src/testnet-demo.ts
```

Project layout

- `api/` - server-side API and services
- `cli/` - CLI tools for asset managers
- `contracts/` - transaction flows and sample XDRs
- `docs/` - protocol design and spec
- `scripts/` - automation and demos
- `src/` - example TypeScript demo code
